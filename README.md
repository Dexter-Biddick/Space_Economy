# Space_Economy
Automated fundamental research and valuation pipeline for 25+ Space Economy tickers using JavaScript (Apps Script) and FMP API. This project automates the ingestion and analysis of fundamental data for 25+ publicly traded space companies.

**Language**: 
JavaScript (Google Apps Script)


**Data Source**: 
Financial Modeling Prep (FMP) API, Jonathan's Space Report


**Automation**: 
Time-based triggers for daily updates


**Key Features**:

  Automated Trading Session Stats - Volume, price, average volume, RSI, VWAP, Sector Volume, pre-market movers

  Volatility Updates - 20 Day High/Low Volume accross all space tickers

  Alternative Data - Tracking launch cadence and constellations from multiple countries and companies

  Institutional Monitor - 13F scraper to identify large buyers/sellers

  Performance Attribution - Update real-time performance (MTD, YTD, Trailing 3mo/12mo/3yr). Ticker level and sector-wide

  Other - Industry breakdown, Launch Calendar, Insider Trade Alerts, House/Senate Trade Alerts, Backlog Scraper using earnings transcripts


**Why This Exists**:
To eliminate manual data entry and build a real-time comprehensive space economy dashboard
